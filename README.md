# システム概要
- aws_experimentはawsを用いた、大学周辺の飲食店マップアプリを構築
- サイドバーには飲食店のジャンルごとにラジオボタンがあり、指定したジャンルのお店のマーカーを出現させることが可能
- 現在地から近い飲食店や、行動履歴からおすすめのお店を表示するレコメンド機能搭載
